О приложении

Приложение "Калькулятор" позволяет выполнять простые математические действия с числами в различных системах счислений (с основанием от 2 до 16).

Управление

Используйте ползунок для выбора текущей системы счисления.

Используйте кнопки:

  • 'MC' - зачистка памяти;
  • 'MR' - вызов числа из памяти;
  • 'M+' - прибавить к числу в памяти;
  • 'M-' - вычесть из числа в памяти;
  • 'MS' - сохранить число из поля ввода в память;
  • '0'-'F' - ввод чисел;
  • '+/-' - получение противоложного числа из текущего;
  • ',' - ввод разделителя;
  • 'CE' - зачистка поля ввода;
  • 'C' - зачистка всех полей;
  • 'BS' - удаление последнего символа в поле ввода;
  • '1/x' - получение обратного числа из текущего;
  • 'x²' - возведение текущего числа в квадрат;
  • '√' - извлечение квадратного корня из текущего числа;
  • '÷' - отношение введённых чисел;
  • '×' - произведение введённых чисел;
  • '-' - разность введённых чисел;
  • '+' - сумма введённых чисел;
  • '=' - получение результата.

Примечание: данные, введённые в поле ввода, будут преобразованы при изменении текущей с. сч.

История

В разделе истории вы получите список совершённых операций за текущую сессию пользования.

Нажатие кнопки "Очистить историю" повлечёт за собой мгновенное удаление истории без возможности восстановления.
